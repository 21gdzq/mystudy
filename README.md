# 我的学习阶段性项目成果
## DeepSeek 文件结构分析Agent

一个命令行工具，用于扫描文件夹结构并使用DeepSeek AI进行分析。

### 安装依赖

```bash
pip install requests python-dotenv
```
### 参数说明
```bash
--path, -p: 要扫描的文件夹路径
--tree, -t: 显示树状结构
--size, -s: 显示文件大小
--json, -j: 保存为JSON文件
--stats: 显示统计信息
--ai: 使用AI分析
--all, -a: 执行所有操作
--depth: AI分析深度
```
